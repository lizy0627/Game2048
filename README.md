功能特性
纯C语言实现：仅使用标准C库和Windows API，无外部依赖
彩色控制台界面：不同数字显示不同颜色，界面美观
完整的游戏逻辑：移动、合并、计分、游戏状态判断
实时数据统计：显示分数、步数和游戏时间
游戏状态判断：自动判断胜利（2048）和失败（无法移动）
菜单系统：开始游戏、规则说明、按键说明
字符艺术：使用ASCII字符绘制精美的标题和状态信息
完整的游戏循环：包含菜单、游戏、结束等完整状态
快速开始
环境要求
操作系统：Windows XP/Vista/7/8/10/11
编译器：支持C99标准的C编译器（如MinGW、MSVC）
构建工具：任何C语言IDE或命令行编译工具
